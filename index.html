2048 Game Information

Pull the tiles with the same value on each other to double up and reach 2048!
Earn points by joining the tiles and beat your best score!
  • Swipe, click or use the arrows/wasd keys on your keyboard
  • Use the menu buttons to restart the current game or choose a different game size
  • With the sound on/off button you can play or stop a song
  • To return to this information, click on the ? button
Little history:

"2048 is a single-player sliding block puzzle game designed by Italian web developer Gabriele Cirulli . 2048 was originally written in JavaScript and CSS during a weekend, and released on March 9, 2014, as free and open-source software subject to the MIT license."
